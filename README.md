# Game Search Web App

A simple web application that allows users to search for games and view basic pricing information.

## Tech Stack

### Frontend
- React
- Vite
- TypeScript

### Backend
- Node.js
- NestJS
- SQLite
- TypeORM

## Features
- List games
- Search games by name (case-insensitive, partial match)
- Clean REST API
- SQL database with seeded data

## Games Included
- FIFA 23
- Red Dead Redemption 2
- Split Fiction
